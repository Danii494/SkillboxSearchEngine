# Search Engine (поисковый движок)

Консольное приложение для поиска по текстовым файлам с ранжированием результатов.

## Стек технологий
- C++17
- Многопоточность (std::thread)
- Библиотека nlohmann/json для работы с JSON
- Google Test для модульного тестирования
- CMake

## Запуск проекта локально

### Требования
- Компилятор с поддержкой C++17 (MinGW, GCC, Clang)
- CMake версии 3.17 или выше

### Сборка
```bash
cd search_engine
mkdir build && cd build
cmake ..
cmake --build .
